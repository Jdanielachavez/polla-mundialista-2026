# 🏆 Polla Mundialista 2026

Aplicación web de quiniela familiar para el **Mundial FIFA 2026**.

## ¿Qué es?
Una página web interactiva donde la familia puede hacer sus predicciones de los partidos del Mundial, acumular puntos y ver el ranking en tiempo real.

## Tecnologías
- HTML · CSS · JavaScript (vanilla)
- Single-page app — sin dependencias externas

## Cómo usarla
Abre `index.html` en cualquier navegador. No requiere instalación ni servidor.

## Archivos
- `index.html` — Aplicación completa
- `fifa2026.png` — Logo del Mundial
- `logo.jpg` — Logo de la polla
- `robots.txt` / `sitemap.xml` — SEO
